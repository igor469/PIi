1. Основная цель проекта
Основная цель проекта - реализация консольной (CLI) программы на Python, которая вычисляет число Pi с точностью N знаков после запятой (задается пользователем в файле параметров) и рассчитывает время в секундах, которое потребуется на вычисление 1 млн знаков числа Pi на этом оборудовании (компьютере-сервере).
Результат расчетов выводится построчно в файл PIi.prn 
В строке выводится L очередных вычисленных знаков числа Pi.

3. Ограничения и допущения проекта
Диапазон возможных значений, который может задать пользователь, для числа знаков числа Pi, которые необходимо вычислить: целое число от 1 до 1000000.
Диапазон возможных значений, который может задать пользователь, для указания количества знаков числа при выводе результата в одной строке PIi.prn: целое число от 1 до 100.

4. Структура, состав входного файла параметров PIi.ini
В файле параметров PIi.ini пользователем указывается 2 параметра:
N - Количество вычисляемых знаков числа Pi после запятой
L - Количество знаков в одной строке вывода в файле PIi.prn
Пример файла PIi.ini
-----
# Comment

[Main]  # Главный-общий раздел параметров
N=1000  # Количество вычисляемых знаков
L=50    # Количество знаков в одной строке вывода
-----

5. Файл журнала (PIi.log) и вывода результатов расчета (PIi.prn)
В файл журнала (PIi.log) выводится следующая информация:
- время старта программы
- время начала     вычислений числа Pi
- время завершения вычислений числа Pi
- время, которое потребуется на вычисление 1 млн знаков числа Pi на этом оборудовании (это время выводится в секундах)
- все ошибки работы
Времена в сообщениях выводятся в формате - чч:мм:сс
Каждая строка журнала имеет вид - 2025-07-10 14:23:27: Сообщение...
Файл журнала дописывается и не удаляется при каждом запуске программы
В файл результатов (PIi.prn) выводится построчно вычисленное число Pi от первого знака после запятой до последнего знака. "3," начало числа Pi не выводится.
